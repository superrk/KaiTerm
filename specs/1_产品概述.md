# 产品概述: KaiTerm

## 产品定位
KaiTerm 是一款桌面端 SSH + SFTP 一体化工具，面向开发者和运维人员，提供高效的远程服务器管理体验。

## 核心价值
- SSH 终端与 SFTP 文件管理无缝集成，共享同一会话
- 支持多 Tab 多服务器连接，提升多机操作效率
- 支持 Windows 本地终端管理

## 目标用户
- 后端开发工程师
- 运维工程师 (SRE/DevOps)
- 需要频繁操作远程 Linux 服务器的技术人员

## 技术栈
- **桌面框架**: Tauri 2.x
- **前端框架**: Vite + Vue 3 + TypeScript
- **SSH/SFTP**: russh + russh-sftp (纯 Rust 异步实现)
- **凭据加密**: 平台原生加密 (Windows DPAPI)

## 平台支持
- 首发: Windows
- 终端: PowerShell / CMD
